## 🧩 第七幕：不定方程式の攻略

不定方程式とは、「未知数の個数より方程式の個数が少ない」方程式のことです。解が無限にあるか、条件を満たす解が有限個かは、絞り込みテクニックで決まります。この幕では、1次・2次・3次不定方程式と、解と係数の関係を活用した攻略法を学びます。

### 🔹 1次不定方程式の基本

**【パターン17】** $3x - 5y = 0$ を満たす整数 $x, y$ を求めよ。

✍️ **思考のステップ**

$$3x = 5y$$

$3 \perp 5$ なので、$3$ は $y$ を割り切り、$5$ は $x$ を割り切ります：

$$x = 5k, \quad y = 3k \quad (k \text{ は整数})$$

💡 **直感イメージ：** 「$3x = 5y$ で $3 \perp 5$」ということは、$x$ は $5$ の倍数、$y$ は $3$ の倍数しかあり得ません。互いに素な数が掛け算で等しいなら、それぞれの変数は相手の係数の倍数になるしかないのです。

---

### 🔹 1次不定方程式：特殊解 → 一般解

**【パターン18】** $100m + 29n = 1$ を満たす整数 $m, n$ を求めよ。

✍️ **思考のステップ（特殊解を1つ見つける → 一般解を導く）**

**Step 1：特殊解を見つける**

ユークリッド互除法の過程を使って、係数を小さくしながら特殊解を探します：

$$100 = 29 \times 3 + 13 \quad \Rightarrow \quad 29(3m+n) + 13m = 1$$

$\square = 3m + n$ とおく：$29\square + 13m = 1$

$$29 = 13 \times 2 + 3 \quad \Rightarrow \quad 13(2\square + m) + 3\square = 1$$

$2\square + m = 1$, $\square = -4$ で満たす → $m = 9$

$100(9) + 29n = 1 \Rightarrow n = -31$

**特殊解：** $(m, n) = (9, -31)$

**Step 2：一般解を導く**

$$100(m - 9) + 29(n + 31) = 0 \quad \Rightarrow \quad 100(m - 9) = -29(n + 31)$$

$100 \perp 29$ なので：$m = -29k + 9, \; n = 100k - 31$（$k$ は整数）

💡 **直感イメージ：** 1次不定方程式の攻略は「特殊解を1つ見つける → 差を取って因数分解 → 互いに素を利用して一般解」という2段構えです。特殊解を見つけるのが難しそうに見えますが、ユークリッド互除法の割り算を逆向きにたどるだけで機械的に求まります。

### 🔹 1次不定方程式：合同式を使う解法

$25m + 17n = 1623$ を満たす整数 $m, n$ を求めよ。

✍️ **思考のステップ（合同式で1変数を消す）**

$25m \equiv 1623 \pmod{17}$

$25 \equiv 8 \pmod{17}$, $1623 \equiv 8 \pmod{17}$

$$8m \equiv 8 \pmod{17}$$

$8 \perp 17$ なので両辺を $8$ で割れる：$m \equiv 1 \pmod{17}$

$m = 1$ を代入：$25 + 17n = 1623 \Rightarrow n = 94$

**一般解：** $m = 17k + 1, \; n = 25k + 94$（$k$ は整数）

> ⚙️ **合同式でも法と互いに素なら両辺を割れる！** これが合同式を使った1次不定方程式解法の核心です。

### 🔹 2次不定方程式：因数分解で絞り込む

**【パターン11】** $\sqrt{n^2 + 200n}$ が整数となる自然数 $n$ をすべて求めよ。

✍️ **思考のステップ**

$k = \sqrt{n^2 + 200n}$ とおく：$(n + 100)^2 - k^2 = 10000$

$$(n + 100 + k)(n + 100 - k) = 10000 = 2^4 \cdot 5^4$$

絞り込み条件：$n \geq 1$, $k \geq 1$, 2数の和は偶数（→ 偶奇が等しい）, 10000が偶数（→ どちらも偶数）

<table>
<tr><th>$n+100+k$</th><th>5000</th><th>2500</th><th>1250</th><th>1000</th><th>500</th><th>200</th></tr>
<tr><th>$n+100-k$</th><th>2</th><th>4</th><th>8</th><th>10</th><th>20</th><th>50</th></tr>
</table>

$n = \frac{(\text{大}) + (\text{小})}{2} - 100$ より：$n = 2401, 1152, 529, 405, 160, 45, 25$

💡 **直感イメージ：** 「平方根が整数」→「2乗して整数」→ 因数分解して「2つの整数の積」→ 約数の組を列挙。2次不定方程式の王道パターンです。

### 🔹 2次不定方程式：解と係数の関係

**【パターン12】** $x^2 - 2(n-1)x - 4n + 3 = 0$ が整数解を持つような $n$ の値を求めよ。

✍️ **思考のステップ**

整数解 $\alpha, \beta$ とおく。解と係数の関係：

$$\alpha + \beta = 2(n-1) \quad \Rightarrow \quad n = \frac{\alpha + \beta}{2} + 1$$

$$\alpha \beta = -4n + 3$$

代入して整理：$(\alpha + 2)(\beta + 2) = 3$

<table>
<tr><th>$\alpha + 2$</th><td>$-1$</td><td>$3$</td></tr>
<tr><th>$\beta + 2$</th><td>$-3$</td><td>$1$</td></tr>
</table>

$n = 1$ または $n = -3$

💡 **直感イメージ：** 解と係数の関係を使うと「$(\alpha + a)(\beta + b) = c$」の因数分解形に持ち込めます。$c$ の約数の組を列挙するだけで全解が手に入ります。

### 🔹 2次不定方程式：判別式が平方数

**【パターン13】** $x^2 + 2nx + 2n^2 + 4n - 16 = 0$ が整数解を持つような $n$ の値を求めよ。

✍️ **思考のステップ**

実数条件：$\frac{D}{4} = -n^2 - 4n + 16 \geq 0 \quad \Rightarrow \quad -6 \leq n \leq 2$

<table>
<tr><th>$n$</th><td>$-6$</td><td>$-5$</td><td>$-4$</td><td>$-3$</td><td>$-2$</td><td>$-1$</td><td>$0$</td><td>$1$</td><td>$2$</td></tr>
<tr><th>$D/4$</th><td>$4$</td><td>$11$</td><td>$16$</td><td>$19$</td><td>$20$</td><td>$19$</td><td>$16$</td><td>$11$</td><td>$4$</td></tr>
</table>

平方数は $4, 16, 16, 4$ → $n = -6, -4, 0, 2$

💡 **直感イメージ：** 2次方程式の整数解条件は「判別式が平方数」。実数条件で範囲を絞り、全数チェックで平方数を探すだけです。

### 🔹 3文字の不定方程式：1変数を固定

**【パターン14】** $x + 2y + 3z = 10$ を満たす自然数 $(x, y, z)$ をすべて求めよ。

✍️ **思考のステップ（1変数を固定して2変数問題に）**

$3z \leq 7$ → $z = 1$ または $z = 2$

* $z = 1$: $x + 2y = 7$ → $(1,3), (3,2), (5,1)$
* $z = 2$: $x + 2y = 4$ → $(2,1)$

**答え：** $(1,3,1), (3,2,1), (5,1,1), (2,1,2)$

💡 **直感イメージ：** 3文字の不定方程式は「1変数の範囲を絞って固定し、残り2変数の問題に帰着させる」のが鉄則です。

### 🔹 3文字の不定方程式：大小関係で絞り込む

**【パターン15】** $0 < x \leq y \leq z$ で $xyz = x + y + z$ を満たす整数を求めよ。

✍️ **思考のステップ**

$3x \leq xyz \leq 3z$ → $3 \leq yz$ と $xy \leq 3$

$(x, y)$ の候補：$(1,1), (1,2), (1,3)$

* $(1,1)$: $z = 2 + z$ → $0 = 2$ ✗
* $(1,2)$: $2z = 3 + z$ → $z = 3$ ✅ → $(1,2,3)$
* $(1,3)$: $3z = 4 + z$ → $z = 2$ ✗（$y \leq z$ 不満）

**答え：$(1, 2, 3)$**

💡 **直感イメージ：** 大小関係で挟むと $xy \leq 3$ という厳しい条件が導かれます。候補が数個しか残りません。

### 🔹 分数型が整数：約数に注目

**【パターン18】** $\dfrac{n^2 + 14n + 33}{3n + 5}$ が整数となる自然数 $n$ を求めよ。

✍️ **思考のステップ**

分母 $3n+5$ が分子を割り切る → 余りを計算すると $3n+5$ は $56$ の約数になる。

$n \geq 1$ → $3n + 5 \geq 8$。$56$ の約数で $\geq 8$：$8, 14, 28, 56$

* $3n+5=8$ → $n=1$
* $3n+5=14$ → $n=3$
* $3n+5=28$ → $n=23/3$ ✗
* $3n+5=56$ → $n=17$

**答え：$n = 1, 3, 17$**

💡 **直感イメージ：** 「分母が分子を割り切る」→「分母が余りの定数を割り切る」に変換。定数の約数を列挙するだけです。

### 🔹 分数型が整数：$k$（整数）とおいて判別式

**【パターン19】** $\dfrac{6x}{x^2 - 3x + 9}$ が整数となる実数 $x$ を求めよ。

✍️ **思考のステップ**

$k = \dfrac{6x}{x^2 - 3x + 9}$ とおく：$kx^2 - (3k+6)x + 9k = 0$

* $k = 0$: $x = 0$
* $k \neq 0$: $D \geq 0$ → $-\frac{2}{3} \leq k \leq 2$ → $k = 1, 2$

$k=1$: $x = \frac{9 \pm 3\sqrt{5}}{2}$, $k=2$: $x = 3$

💡 **直感イメージ：** 分数型が整数で分子の次数 ≥ 分母の次数のとき、$= k$（整数）とおいて2次方程式に変換し、判別式で $k$ の範囲を絞るのが王道です。

### 🔹 【パターン16】素数に関する不定方程式

例題： $n, n+2, n+4$ がすべて素数なのは $n = 3$ の場合だけであることを示せ。

✍️ **思考のステップ**

$n \equiv 0, 1, 2 \pmod{3}$ で分類：

<table>
<tr><th></th><th>$n \equiv 0$</th><th>$n \equiv 1$</th><th>$n \equiv 2$</th></tr>
<tr><th>$n$</th><td>$0$</td><td>$1$</td><td>$2$</td></tr>
<tr><th>$n+2$</th><td>$2$</td><td>$0$</td><td>$1$</td></tr>
<tr><th>$n+4$</th><td>$1$</td><td>$2$</td><td>$0$</td></tr>
</table>

どのケースでも $n, n+2, n+4$ のいずれかが $3$ の倍数。$n \geq 4$ なら $3$ より大きい $3$ の倍数は素数でない。$n = 3$ のときのみ $3$ 自身が素数。 ✓

💡 **直感イメージ：** 3つの数は必ずどれかが3の倍数。それが素数であるには「3自身」しかあり得ない、というロジックです。

### 🔹 【パターン27】任意の $n$ で整数となる条件

例題： $a, b$ を自然数とする。任意の自然数 $n$ に対して $\dfrac{an+b}{n^2+1}$ が整数となる $a, b$ の値を求めよ。

✍️ **思考のステップ（具体的な数値を代入して絞り込む）**

$n = 1$: $\dfrac{a+b}{2}$ が整数

$n = 2$: $\dfrac{2a+b}{5}$ が整数

$n = 3$: $\dfrac{3a+b}{10}$ が整数

$n=1$ と $n=2$ の条件から、$b=1$ でないと全ての $n$ で整数にならないことを確認。

$b = 1$ を代入：$n=1$ で $\dfrac{a+1}{2}$ が整数 → $a$ は奇数

$n=2$ で $\dfrac{2a+1}{5}$ が整数 → $2a+1 \equiv 0 \pmod{5}$ → $a \equiv 2 \pmod{5}$

$a = 1$ のとき全ての $n$ で確認 → 整数となる。

**答え：$(a, b) = (1, 1)$**

💡 **直感イメージ：** 「任意の $n$ で整数」の問題は、具体的な $n$ を代入して条件を積み重ねるのが鍵です。$n = 1, 2, 3$ などに代入して得られた条件の共通部分を取ります。
