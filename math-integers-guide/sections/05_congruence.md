## ⚖️ 第五幕：合同式と連続整数の積

「余りの世界」で計算する道具が**合同式**です。合同式を使うと、巨大な数の余りも一瞬で計算でき、連続整数の積がどの倍数になるかも機械的に判定できます。

### 🔹 合同式とは

$$a \equiv b \pmod{p}$$

「$a$ と $b$ は $p$ を法として合同」＝ **$a$ と $b$ を $p$ で割った余りが等しい**

例：$-3 \equiv 2 \pmod{5}$（$-3$ に $5$ を足すと $2$ になる）

---

### 🔹 合同式の基本性質

$a \equiv b \pmod{p}$ のとき：

* **加法：** $a + c \equiv b + c \pmod{p}$
* **減法：** $a - c \equiv b - c \pmod{p}$
* **乗法：** $a \times c \equiv b \times c \pmod{p}$
* **累乗：** $a^m \equiv b^m \pmod{p}$

> ⚠️ **注意！** **除法は注意が必要！** $c \perp p$（$c$ と $p$ が互いに素）のときのみ、$a \times c \equiv b \times c \pmod{p}$ の両辺を $c$ で割って $a \equiv b \pmod{p}$ が成り立ちます。$c$ と $p$ が互いに素でない場合は、両辺を割ってはいけません。

💡 **直感イメージ：** 合同式は「余りの世界で同じ値を見ている」という意味です。$11 \equiv 1 \pmod{5}$ なら、$11$ も $1$ も「5で割ると1余る仲間」。だから $11^{100}$ も $1^{100} = 1$ と同じ仲間になるのです。

---

### 🔹 合同式の計算例

**例1：** $11^{100}$ を $5$ で割った余りは？

$$11 \equiv 1 \pmod{5} \quad \Rightarrow \quad 11^{100} \equiv 1^{100} = 1 \pmod{5}$$

**答え：1**

**例2：** $n$ を $5$ で割った余りが $3$ のとき、$n^2 + 5n + 2$ を $5$ で割った余りは？

$$n \equiv 3 \pmod{5}$$

$$n^2 + 5n + 2 \equiv 3^2 + 5 \times 3 + 2 = 9 + 15 + 2 = 26 \equiv 1 \pmod{5}$$

**答え：1**

💡 **直感イメージ：** 合同式では「$5n$」の項は $5$ の倍数なので $\equiv 0 \pmod{5}$ で消えてしまいます。だから $n^2 + 5n + 2 \equiv n^2 + 0 + 2 \pmod{5}$ とシンプルになります。余りの世界では、法の倍数はすべて「ゼロ」になるのです。

---

### 🔹 余りによる分類

整数を $p$ で割った余りで分類すると、すべての整数は $p$ 種類に分類されます。

**例：3の倍数判定の分類**

<table>
<tr><th>余り</th><th>$n$</th><th>$n+1$</th><th>$n+2$</th></tr>
<tr><td>0</td><td>$3k$</td><td>$3k+1$</td><td>$3k+2$</td></tr>
<tr><td>1</td><td>$3k+1$</td><td>$3k+2$</td><td>$3k$</td></tr>
<tr><td>2</td><td>$3k+2$</td><td>$3k$</td><td>$3k+1$</td></tr>
</table>

余りが $0, 1, 2$ の3通りに分類すると、連続3整数 $n, n+1, n+2$ の中には必ず3の倍数（余り0）が1つ含まれます。

💡 **直感イメージ：** 整数を「余りという色」で塗り分けると、連続整数は「色が1つずつズレていく」のがわかります。3色で塗れば、連続3整数は必ず3色すべてを網羅します。だから必ず3の倍数が含まれるのです。

---

### 🔹 連続整数の積が作る倍数

連続する整数の積は、必ず特定の倍数になります：

* **連続2整数の積：** $n(n+1)$ → **2の倍数**（どちらかが偶数）
* **連続3整数の積：** $n(n+1)(n+2)$ → **6の倍数**（2の倍数 かつ 3の倍数）
* **連続4整数の積：** $n(n+1)(n+2)(n+3)$ → **24の倍数**（2, 3, 4の倍数）

💡 **直感イメージ：** 連続3整数 $n, n+1, n+2$ を並べると、余りが $0, 1, 2$ のどれかを1つずつ必ず含みます。余り0（3の倍数）が必ず1つ、余り0または2（2の倍数）が必ず1つ。だから積は $2 \times 3 = 6$ の倍数になるのです。

---

### 🔹 【パターン26】素数の平方から1を引いた数

例題： 自然数を $6$ の剰余で分類することにより、$2$ と $3$ 以外の素数 $p$ について $p^2 - 1$ が $24$ の倍数になることを証明せよ。

✍️ **解法1：$p = 6k \pm 1$ とおく**

$2$ と $3$ 以外の素数 $p$ は、$6$ で割ると余りが $1$ または $5$（$= -1$）になります（余り $0, 2, 3, 4$ だと $2$ または $3$ の倍数になってしまう）。

$$p = 6k \pm 1$$

$$p^2 - 1 = (6k \pm 1)^2 - 1 = 36k^2 \pm 12k = 12k(3k \pm 1)$$

* $k$ が偶数のとき：$12k$ は $24$ の倍数 → $p^2 - 1$ は $24$ の倍数
* $k$ が奇数のとき：$(3k \pm 1)$ が偶数（奇数 $\times 3 \pm 1$ = 偶数）→ $12 \times 2 = 24$ の倍数

よって $p^2 - 1$ は常に $24$ の倍数。 ✓

✍️ **解法2：合同式で分ける**

$p$ が $2$ 以外の素数 → $p$ は奇数 → $p \equiv 1, 3, 5, 7 \pmod{8}$

どの奇数を2乗しても $p^2 \equiv 1 \pmod{8}$ → $p^2 - 1 \equiv 0 \pmod{8}$

$p$ が $3$ 以外の素数 → $p \not\equiv 0 \pmod{3}$ → $p^2 \equiv 1 \pmod{3}$ → $p^2 - 1 \equiv 0 \pmod{3}$

$p^2 - 1$ は $8$ の倍数 かつ $3$ の倍数 → $\text{GCD}(8,3) = 1$ より $24$ の倍数。 ✓

💡 **直感イメージ：** 「$p^2 - 1 = (p-1)(p+1)$」と因数分解すると、これは「$p$ の両隣の偶数2つの積」です。奇数 $p$ の両隣は2つの連続する偶数なので、片方は必ず $4$ の倍数、もう片方は $2$ の倍数。積は $8$ の倍数。さらに $p$ が $3$ の倍数でなければ、$p-1, p, p+1$ の連続3整数のうち $p$ 以外のどちらかが $3$ の倍数。だから $24 = 8 \times 3$ の倍数になるのです。

---

### 🔹 倍数の間隔の直感

* **2の倍数：** 1つおき（2つに1つ）
* **3の倍数：** 2つおき（3つに1つ）
* **4の倍数：** 3つおき（4つに1つ）

この「間隔」の感覚が、連続整数の積がなぜ特定の倍数になるかの直感的な理由です。連続 $k$ 整数の中には、$k$ の倍数が必ず1つ含まれます。
