## 🔢 第一幕：倍数の見分け方〜余りに注目する

「ある数が何の倍数か」を見分ける問題は、整数の世界への入門です。ここで鍵になるのは**「余りに注目する」**という発想です。4桁の数 $\overline{abcd}$ を例に、各桁の数字がどう関係しているのかを解き明かしましょう。

### 🔹 倍数判定の基本原理：展開して「倍数のブロック」を見つける

4桁の自然数 $\overline{abcd}$ は、次のように展開できます。

$$\overline{abcd} = 1000a + 100b + 10c + d$$

この展開式の各項（$1000, 100, 10$ など）が、判定したい倍数の**倍数であるかどうか**をチェックすることで、「どの部分に注目すればよいか」が決まります。

---

### 🔹 2の倍数・4の倍数・5の倍数（末尾に注目）

**2の倍数：** $1000, 100, 10$ はいずれも2の倍数です。したがって、$1000a + 100b + 10c$ は必ず2の倍数になります。残るは $d$ のみ。

$$\overline{abcd} = \underbrace{(1000a + 100b + 10c)}_{\text{常に2の倍数}} + d$$

⇒ $d$ が2の倍数（0, 2, 4, 6, 8）なら、$\overline{abcd}$ も2の倍数

**4の倍数：** $1000$ と $100$ は4の倍数ですが、$10$ は4の倍数ではありません。したがって、$10c + d$（下2桁）に注目します。

$$\overline{abcd} = \underbrace{(1000a + 100b)}_{\text{常に4の倍数}} + (10c + d)$$

⇒ $10c + d$（下2桁）が4の倍数なら、$\overline{abcd}$ も4の倍数

**5の倍数：** $1000, 100, 10$ はいずれも5の倍数です。残るは $d$ のみ。

⇒ $d$ が5の倍数（$d = 0$ または $d = 5$）なら、$\overline{abcd}$ も5の倍数

💡 **直感イメージ：** 大きな桁（千の位、百の位など）は、自動的に「倍数のブロック」としてまとまって消えてしまいます。残った「末尾の小さな部分」だけが判定の鍵を握っているのです。2と5は「10の素因数」だからこそ、末尾だけで判定できるのです。

---

### 🔹 3の倍数・9の倍数（各位の和に注目）

3と9の倍数判定は、2や5とは全く異なる仕組みを持っています。ここが整数の世界で最も美しい構造の一つです。

**3の倍数：** 展開式を巧みに変形します。

$$\overline{abcd} = 1000a + 100b + 10c + d = (999+1)a + (99+1)b + (9+1)c + d$$

$$= \underbrace{(999a + 99b + 9c)}_{\text{常に3の倍数}} + (a + b + c + d)$$

$999, 99, 9$ はいずれも3の倍数なので、第1項は常に3の倍数です。残るは各位の和 $a + b + c + d$ だけです。

⇒ 各位の和 $a + b + c + d$ が3の倍数なら、$\overline{abcd}$ も3の倍数

**9の倍数：** まったく同じ変形を使います。$999, 99, 9$ はいずれも9の倍数でもあります。

$$= \underbrace{(999a + 99b + 9c)}_{\text{常に9の倍数}} + (a + b + c + d)$$

⇒ 各位の和 $a + b + c + d$ が9の倍数なら、$\overline{abcd}$ も9の倍数

💡 **直感イメージ：** $1000 = 999 + 1$ という分解の「1」の部分こそが、各位の数字 $a$ をそのまま残します。つまり、$1000a$ の中には「$a$ 個分の9（の倍数）」と「生の $a$」が隠れています。これがすべての桁で起きるから、最終的には「各位の数字を全部足す」だけで9（または3）の倍数判定ができるのです。これは数学の魔術としか言いようがありません！

### 🔹 12の倍数（合成数の判定）

**12の倍数：** $12 = 3 \times 4$ であり、3と4は互いに素です。

⇒ 12の倍数 = 3の倍数 **かつ** 4の倍数

💡 **直感イメージ：** $12 = 3 \times 4$ のように互いに素な因数に分解できる数の倍数判定は、「両方の条件を同時に満たす」というAND条件に置き換えられます。3の倍数は各位の和、4の倍数は下2桁で判定できるので、2つの異なるチェックを組み合わせるだけです。

---

### 🔹 【パターン1】不明な数が複数ある：9の倍数の絞り込み

例題： 6桁の自然数 $\overline{92\square85\triangle}$ が9の倍数になるのは何通りか？

✍️ **思考のステップ（各位の和で絞り込む）**

Step 1：各位の和を計算する

$$9 + 2 + \square + 8 + 5 + \triangle = 24 + \square + \triangle$$

これが9の倍数になれば条件を満たします。

Step 2：$\square + \triangle$ の範囲を絞り込む

各桁の数字は $0 \sim 9$ なので、$0 \leq \square + \triangle \leq 18$ です。したがって：

$$24 \leq 24 + \square + \triangle \leq 24 + 18 = 42$$

Step 3：24から42の間で9の倍数を探す

9の倍数は $27$ と $36$ の2つです。

**(ア) $24 + \square + \triangle = 27$ のとき（$\square + \triangle = 3$）**

$(\square, \triangle) = (0,3), (1,2), (2,1), (3,0)$ ➔ **4通り**

**(イ) $24 + \square + \triangle = 36$ のとき（$\square + \triangle = 12$）**

$(\square, \triangle) = (3,9), (4,8), (5,7), (6,6), (7,5), (8,4), (9,3)$ ➔ **7通り**

**合計：** $4 + 7 = 11$通り

💡 **直感イメージ：** 不明な数の和の範囲（0〜18）を先に確定させ、その中で条件を満たす9の倍数を「候補」としてピックアップする。あとは各候補について、桁の数字の組合せを数え上げるだけです。この「範囲 → 候補 → 数え上げ」の3段階絞り込みは、整数問題の基本パターンです。

### 🔹 【パターン2】45の倍数：複数条件の絞り込み

例題： 6桁の自然数 $\overline{92\square85\triangle}$ が45の倍数になるのは何通りか？

✍️ **思考のステップ（条件を分解して重ねる）**

Step 1：$45 = 9 \times 5$（互いに素）なので、45の倍数 = 9の倍数 **かつ** 5の倍数

Step 2：パターン1の結果（9の倍数となる11通りの組合せ）の中から、5の倍数条件を適用

5の倍数条件は「1の位が0または5」、つまり $\triangle = 0$ または $\triangle = 5$ です。

パターン1の組合せの中で、$\triangle = 0$ または $\triangle = 5$ になるものを探します：

* $(\square, \triangle) = (3, 0)$ ➔ $\triangle = 0$ ✅
* $(\square, \triangle) = (7, 5)$ ➔ $\triangle = 5$ ✅

**答え：2通り**

💡 **直感イメージ：** 合成数の倍数判定は「フィルターを重ねる」感覚です。まず9の倍数というフィルターで11通りに絞り、次に5の倍数というフィルターを重ねて2通りにまで絞り込みます。条件を分解して段階的に適用することで、見落としを防げます。

### 🔹 【パターン3】分数形が整数となる条件

例題： $\dfrac{3^2 - n}{7}$ が自然数となる自然数 $n$ をすべて求めよ。

✍️ **思考のステップ（範囲を絞り込む）**

$\dfrac{3^2 - n}{7}$ が自然数ということは、$3^2 - n = 7k$（$k$ は自然数）とおけます。

$$n = 9 - 7k$$

$n$ は自然数（$n \geq 1$）なので：

$$9 - 7k \geq 1 \quad \Rightarrow \quad 7k \leq 8 \quad \Rightarrow \quad k \leq \frac{8}{7}$$

$k$ は自然数なので $k = 1$ のみ。

$$n = 9 - 7 \times 1 = 2$$

💡 **直感イメージ：** 「分数が整数」という条件は、「分子が分母の倍数」と言い換えられます。さらに「自然数」という条件（$\geq 1$）が範囲を絞り込む強力な制約になります。範囲を確定してから候補を数えるのが鉄則です。

---

### 🔹 分数形が整数となる条件（素因数の指数アプローチ）

もう一つの強力なアプローチは、**素因数の指数を文字に置き換えて範囲を絞り込む**方法です。

**例：** $n = 2^a \cdot 3^b$ と表せる自然数 $n$ について、$\dfrac{n^2}{72}$ が正の整数となる条件を求めよ。

✍️ **思考のステップ**

$$n^2 = 2^{2a} \cdot 3^{2b}, \quad 72 = 2^3 \cdot 3^2$$

$$\dfrac{n^2}{72} = \dfrac{2^{2a} \cdot 3^{2b}}{2^3 \cdot 3^2} = 2^{2a-3} \cdot 3^{2b-2}$$

これが正の整数になるには、指数がすべて非負、つまり：

$$2a - 3 \geq 0 \quad \text{かつ} \quad 2b - 2 \geq 0$$

$$\Rightarrow a \geq 2 \quad \text{かつ} \quad b \geq 1$$

💡 **直感イメージ：** 素因数分解して指数を比較するのは、「成分ごとに在庫をチェックする」感覚です。分母の「2が3個必要」に対して、分子が「2が $2a$ 個ある」。$2a \geq 3$ つまり $a \geq 2$ で在庫が足りる、という判定です。
