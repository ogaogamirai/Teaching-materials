## 🕊️ 第八幕：素数・鳩の巣原理・小数

最後の幕では、整数論の多彩なトピックを扱います。素数の性質、鳩の巣原理による存在証明、有限小数・循環小数の条件、そしてモニック多項式の整数解について学びます。

### 🔹 素数とは

素数とは、**2以上の整数で、1と自分自身以外の正の約数を持たない数**です。2以外の素数はすべて奇数です。

**素数の性質：**

* $(n-7)(n-13) = P$（$P$ は素数）のとき、「$n-7 = \pm P$ かつ $n-13 = \pm 1$」または「$n-7 = \pm 1$ かつ $n-13 = \pm P$」
* 素数でないことを示すには、「2以上の整数の積で表せる」ことを示せばよい
* 因数分解のテクニック：$n^4 + 4 = (n^2+2+2n)(n^2+2-2n)$（ソフィ・ジェルマンの恒等式）

💡 **直感イメージ：** 素数は「これ以上分解できない原子」です。2つの整数の積が素数になるとしたら、片方は必ず $\pm 1$（分解しても意味のない「単位」）でなければなりません。

### 🔹 鳩の巣原理

**原理：** $n$ 個の巣に $n+1$ 羽の鳩が入るとき、**2羽以上の鳩が入る巣が少なくとも1つ存在する**。

💡 **直感イメージ：** 4つの箱に5つのボールを入れたら、どこかの箱に必ず2つ以上入ります。「平均すれば1つずつ」でも、全部が1つずつにはならない、という当たり前の事実が強力な証明ツールになります。

### 🔹 【パターン24】格子点の中点問題

例題： 互いに異なる5個の格子点を任意に選ぶと、その中に中点も格子点となるような対が少なくとも1組存在することを証明せよ。

✍️ **思考のステップ（鳩の巣原理で存在を示す）**

格子点 $(a, b)$ の中点が格子点になる条件は、$a + c$ と $b + d$ がともに偶数であること（$(a,b)$ と $(c,d)$ の偶奇が同じ）。

格子点を「座標の偶奇」で4種類に分類します：

<table>
<tr><th>巣（パターン）</th><td>(偶, 偶)</td><td>(奇, 偶)</td><td>(偶, 奇)</td><td>(奇, 奇)</td></tr>
</table>

5個の格子点（鳩）を4種類の巣に入れる → 鳩の巣原理により、同じパターンの格子点が少なくとも2個存在する。

同じパターンの2点は偶奇が一致する → 和が偶数 → 中点が格子点。 ✓

💡 **直感イメージ：** 格子点の「偶奇の組合せ」は4通りしかありません。5つの格子点を4つの巣に分類すれば、必ず同じ巣に2つ入ります。同じ巣＝同じ偶奇＝和が偶数＝中点が格子点、という美しい連鎖です。

### 🔹 有限小数になる条件

**定理：** 分数 $\dfrac{a}{b}$（既約分数）が有限小数で表せる ⟺ **分母 $b$ の素因数が2と5のみ**

**例：** $\dfrac{1}{8} = 0.125$（$8 = 2^3$）→ 有限小数 ✅

$\dfrac{1}{3} = 0.333\ldots$（$3$ は2でも5でもない）→ 循環小数

**応用：** $\dfrac{N}{2^k \cdot 5^r}$ の形の分数は有限小数です。分母を10の倍数に変形できるからです。

💡 **直感イメージ：** 有限小数 = 「分母を10の累乗にできる」=「分母が $2^a \cdot 5^b$」。10 = 2 × 5 だから、分母が2と5だけでできていれば、適切な2と5を掛けて10の累乗に変形できます。

### 🔹 【パターン21】小数第三位まで表せる条件

例題： $\dfrac{N}{2^3 \cdot 3^3 \cdot 5^3}$ が小数第三位までの小数で表せる自然数 $N$ はいくつあるか？

✍️ **思考のステップ**

「小数第三位までの小数」= $1000$ 倍すると整数になる：

$$\frac{1000N}{2^3 \cdot 3^3 \cdot 5^3} = \frac{1000N}{27000} = \frac{N}{27} \text{ が整数}$$

$1000 = 2^3 \cdot 5^3$ なので、分母の $2^3 \cdot 5^3$ が約分されて消えます：

$$\frac{N}{3^3} = \frac{N}{27} \text{ が整数}$$

→ $N$ は $27$ の約数。$27 = 3^3$ の正の約数は $(3+1) = 4$ 個。

**答え：4個**

💡 **直感イメージ：** 「小数第 $k$ 位まで」=「$10^k$ 倍すると整数」。分母の $2$ と $5$ は $10^k$ で約分されて消え、残った素因数の約数の個数が答えになります。

### 🔹 循環小数

循環小数は分数に直せます：

$$0.123123\ldots = 0.\dot{1}2\dot{3} = \frac{123}{999} = \frac{41}{333}$$

$$2.8333\ldots = 2.8\dot{3} = 2 + \frac{8}{10} + \frac{3}{90} = \frac{17}{6}$$

> ⚙️ **循環節の長さが $n$ のとき、$10^n - 1$ で割ることで分数に直せる。**

### 🔹 モニック多項式の有理数解

**【パターン22】** 最高次係数が1の多項式（モニック多項式）$x^3 + ax^2 + bx + c = 0$（$a, b, c$ は整数）が有理数解を持つならば、その解は整数であることを示せ。

✍️ **思考のステップ**

有理数解を $\dfrac{p}{q}$（$p \perp q$, $p$ は自然数, $q$ は整数）とおく。$q = 1$ を示す。

方程式に代入：

$$\left(\frac{p}{q}\right)^3 + a\left(\frac{p}{q}\right)^2 + b\left(\frac{p}{q}\right) + c = 0$$

両辺に $q^2$ を掛ける：

$$\frac{p^3}{q} + ap^2 + bpq + cq^2 = 0$$

右辺の $ap^2 + bpq + cq^2$ は整数。したがって $\frac{p^3}{q}$ も整数。

$p \perp q$ で $\frac{p^3}{q}$ が整数 → $q = 1$。

よって解 $\frac{p}{q} = p$（整数）。 ✓

💡 **直感イメージ：** モニック多項式（最高次係数が1）では、分母 $q$ が「約分できない（$p \perp q$）」にもかかわらず「$q$ で割り切れる」必要があります。これが成り立つのは $q = 1$ のときだけ。つまり有理数解は必ず整数解になります。

### 🔹 【パターン23】有理根定理（一般の多項式）

**定理：** $ax^4 + bx^3 + cx^2 + dx + e = 0$（$ae \neq 0$, $a, b, c, d, e$ は整数）が有理数解 $\frac{p}{q}$（$p \perp q$）を持つとき：

* **$p$ は定数項 $e$ の約数**
* **$q$ は最高次係数 $a$ の約数**

✍️ **証明の流れ**

方程式に $\frac{p}{q}$ を代入し、両辺に $q^3$ を掛けると：

$$ap^4 + bp^3q + cp^2q^2 + dpq^3 + eq^4 = 0$$

これを $p$ の約数として整理：

$$p(ap^3 + bp^2q + cpq^2 + dq^3) = -eq^4$$

左辺は $p$ の倍数 → $p$ は $eq^4$ の約数。$p \perp q$ → $p$ は $e$ の約数。

同様に $q$ で整理すると、$q$ は $a$ の約数。

💡 **直感イメージ：** 有理根定理は「分子は定数項の約数、分母は最高次係数の約数」というシンプルなルールです。モニック多項式（$a = 1$）のとき、分母は $1$ の約数 = $1$ しかないので、解が必ず整数になるのです。
