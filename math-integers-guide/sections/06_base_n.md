## 🔟 第六幕：n進数の世界

私たちが普段使っているのは10進数ですが、コンピュータは2進数、時間は60進数を使います。n進数の世界では「何を1つのまとまりとするか」が変わるだけで、本質的な考え方は同じです。

### 🔹 n進数の基本

10進数 $1023_{(10)}$ は次のように展開されます：

$$1023_{(10)} = 1 \times 10^3 + 0 \times 10^2 + 2 \times 10^1 + 3 \times 10^0$$

一般に、$n$進数では各桁の数値は $0 \sim n-1$ の範囲です。

💡 **直感イメージ：** 10進数は「10個で1つの袋にまとめる」世界。2進数は「2個で1つの袋にまとめる」世界。袋の大きさが変わるだけで、中身を数える行為は同じです。

---

### 🔹 n進数への変換（整数部分）：割り算で下から読む

**【パターン9】** $78_{(10)}$ を $5$進数に変換せよ。

✍️ **思考のステップ（底で割り続ける）**

```
78 ÷ 5 = 15 余り 3
15 ÷ 5 =  3 余り 0
 3 ÷ 5 =  0 余り 3
```

下から読み上げて：$78_{(10)} = 303_{(5)}$

確認：$3 \times 5^2 + 0 \times 5^1 + 3 \times 5^0 = 75 + 0 + 3 = 78$ ✓

💡 **直感イメージ：** 「5進数に変換する」=「5個ごとに袋に詰める」ことです。78個のボールを5個ずつ袋に詰めると15袋と余り3個。さらに15袋を5袋ずつ大きな箱に詰めると3箱と余り0袋。余りを下から読むと「303」になるのです。

---

### 🔹 n進数への変換（小数部分）：掛け算で上から読む

**【パターン10】** $0.776_{(10)}$ を $5$進数に変換せよ。

✍️ **思考のステップ（底を掛け続ける）**

```
0.776 × 5 = 3.880  → 整数部分 3
0.880 × 5 = 4.400  → 整数部分 4
0.400 × 5 = 2.000  → 整数部分 2
```

上から読んで：$0.776_{(10)} = 0.342_{(5)}$

確認：$3 \times 5^{-1} + 4 \times 5^{-2} + 2 \times 5^{-3} = 0.6 + 0.16 + 0.016 = 0.776$ ✓

💡 **直感イメージ：** 小数部分の変換は「底を掛けて整数部分を取り出す」の繰り返しです。「5進数の小数第1位」は $\frac{1}{5}$ の位なので、元の数に5を掛けると整数部分に「5進数の小数第1位」が飛び出してきます。

---

### 🔹 n進数の桁数と範囲

$n$進数で $k$ 桁の自然数 $N$ は以下の範囲に入ります：

$$n^{k-1} \leq N < n^k$$

**例：** 2進数8桁 → $2^7 \leq N < 2^8$ → $128 \leq N < 256$

3進数6桁 → $3^5 \leq N < 3^6$ → $243 \leq N < 729$

💡 **直感イメージ：** $n$進数 $k$ 桁の最大値は「各桁がすべて $n-1$」= $n^k - 1$ です。最小値は「最上位桁が1、それ以外が0」= $n^{k-1}$ です。

### 🔹 n進数の絞り込み問題

例題： 自然数 $N$ は2進数で8桁でその下2桁が「01」、3進数で6桁でその下2桁が「02」である。$N$ を求めよ。

✍️ **思考のステップ（範囲を確定 → 条件を重ねる）**

**Step 1：桁数から範囲を絞る**

2進数8桁：$128 \leq N < 256$

3進数6桁：$243 \leq N < 729$

両方を満たす範囲：$243 \leq N \leq 255$ …①

**Step 2：3進数で下2桁「02」の条件**

3進数で下2桁が「02」= $N$ を $9$（$= 3^2$）で割った余りが $2$：

$$N \equiv 2 \pmod{9}$$

①の範囲 $243 \sim 255$ で $N \equiv 2 \pmod{9}$ を満たすもの：$245$ と $254$

**Step 3：2進数で下2桁「01」の条件**

2進数で下2桁が「01」= $N \equiv 1 \pmod{4}$：

* $245 \equiv 1 \pmod{4}$ ✅
* $254 \equiv 2 \pmod{4}$ ✗

**答え：$N = 245$**

💡 **直感イメージ：** n進数の「下 $k$ 桁」の条件は、「$n^k$ で割った余り」と等価です。3進数の下2桁「02」= $9$ で割った余りが $2$。2進数の下2桁「01」= $4$ で割った余りが $1$。この「桁の条件 → 合同式への翻訳」がn進数絞り込みの鍵です。

---

### 🔹 n進数の各桁の数字の範囲

$n$進数の各桁の数字は $0 \sim n-1$ の範囲ですが、最上位桁だけは $1 \sim n-1$（0は不可）です。

**例：** $N = \overline{abc}_{(6)}$ かつ $N = \overline{cab}_{(9)}$ のとき

* 6進数の条件：$1 \leq a \leq 5$, $0 \leq b, c \leq 5$
* 9進数の条件：$1 \leq c \leq 8$, $0 \leq a, b \leq 8$

両方を満たす：$1 \leq a, c \leq 5$, $0 \leq b \leq 5$

💡 **直感イメージ：** 複数のn進数表記があるとき、「各桁の数字」がすべての進数で有効な範囲に入っていなければなりません。より厳しい方の進数の範囲が制約となります。
