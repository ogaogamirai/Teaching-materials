# 数学「場合の数」直感ビジュアル学習アプリ

本アプリは、高校数学で学ぶ「場合の数」の様々な計算と可視化を直感的に体験できるインタラクティブ学習教材です。
数式（$nPr$, $nCr$, $nHr$ など）の計算結果だけでなく、実際にモノがどのように並ぶのか、どのようにグループ分けされるのかを視覚的にアニメーションで確認できます。

---

## 📂 収録されている学習テーマ

### 1. 並べる（順列）
- **基本の順列 ($nPr$)**: 異なるモノを並べ替える基本的な順列。
- **同じものがある順列**: 重複する要素がある場合の並べ替え。区別をなくすための「重複度による割り算」をビジュアルで説明します。
- **円順列 / 同じものがある円順列**: 円環状に並べる際の、回転による重なり（周期性・例外周期）を可視化。
- **数珠順列**: ネックレスのように裏返して同じになるパターンの重複調整（左右対称性の考慮）。
- **隣接制限**: 「特定の要素が隣り合う（1つにまとめる）」「隣り合わない（隙間に挿入する）」条件の視覚化。

### 2. 選ぶ（組合せ）
- **基本の組合せ ($nCr$)**: 順序を考慮せずに選ぶ組み合わせ。
- **組分け（グループ分け）**: メンバーをいくつかのグループに分ける方法。要素数が同じグループ同士の「組の区別」による重複調整。
- **部屋割り**: 空き部屋の有無、部屋の名前（区別）があるかないかによる場合の数の違いをインタラクティブに比較。

### 3. 重複組合せ（仕切り棒モデル）
- **重複組合せ ($nHr$)**: 「りんごと仕切り」を用いたモデル。もらえない人がいても良い配分と、最低個数制限（あらかじめ配っておく）の比較。

### 4. 組合せ相関（写像12相の一部）
- 「モノの区別（あり/なし）」×「グループの区別（あり/なし）」のマトリックスによって、問題がどのように変化するかを相関図で直感的に理解します。

### 5. 完全順列（攪乱順列）
- 席替え問題など、どの要素も元の位置に戻らない並べ方。漸化式 $W(n) = (n-1)\{W(n-1) + W(n-2)\}$ のアニメーション可視化。

## 🛠️ 技術スタックと構成
- **Core**: HTML5, JavaScript (Vanilla ES6+)
- **Styling**: CSS3 (モダンなダークスペーステーマ、滑らかなCSS Transitions)
- **可視化**: DOMアニメーション & Canvas描画

## 🚀 起動方法
本フォルダ内の `index.html` をブラウザで開くだけで起動します（特別なサーバーは不要です）。
あるいは、ローカルサーバーを起動してアクセスしてください。
